# Rock Paper Scissors


By Baraffe Robin (https://twitter.com/RobinBaraffe).


## Description
**Rock Paper Scissors** 
Simple game in command line. Options can be 0 : Rock, 1 : Paper, 2 : Scissors.

## Installation

Clone the repository and run

```go
go build
```

Then, run the .exe

```console
./main
```
